  • Tools On Hold

    • Tap Sequence Detector
      • Detect "N" (hold or release)taps and "T" time held after the Taps per Button to Trigger Commands in your Games
        EX: Quandruple Tap + Hold for 2.5 seconds = CMD, Single Tap + Release for 1 seconds = CMD
    • Button Combo Detector
      • Using the "Tap Sequence Detector" above we can gather and inspect multiple tap sequences for multiple buttons to detect Button Combos
        EX: up -> up -> down -> down -> left -> right -> left -> right -> B -> A = CMD
    • Selection Tool (In Game)
      • Select objects like you would in the windows file explorer OR use the custom Lasso/Polygonal Lasso Tool to select many objects at once.

    • Polygon Edit Tool (In Game)
      • Add or Remove a polygon, Add or Remove the Vertices of your polygon, Shift the Points of your polygon, and Drag the edges of your polygon
        I will (Expand && Refine) the implementation described Here
